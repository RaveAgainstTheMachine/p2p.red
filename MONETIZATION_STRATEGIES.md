### 2. **Premium Tier (Freemium Model)**
Keep core P2P free, charge for convenience features.

**Free Tier:**
- File size: 5GB max
- Link expiry: 24 hours
- Basic P2P transfer
- Standard STUN servers

**Premium Tier ($5-10/month):**
- File size: 50GB max
- Link expiry: 7 days
- Priority TURN servers (better NAT traversal)
- Custom branded share pages
- Batch transfers (multiple files at once)
- Transfer history (sender only, privacy-preserving)
- Password protection with custom PINs
- Email notifications when transfer completes
- Remove donation prompts

**Revenue Potential:** $500-5000/month (100-500 paying users)

### 3. **Business/Team Plans**
Target small businesses and teams.

**Team Plan ($20-50/month):**
- Everything in Premium
- Team workspace (shared links)
- Admin dashboard
- Usage analytics (aggregated, no personal data)
- Custom domain (files.yourcompany.com)
- White-label option
- Priority support
- SLA guarantee (99.9% uptime)

**Revenue Potential:** $1000-10000/month (20-200 teams)

**Revenue Potential:** $500-5000/month
